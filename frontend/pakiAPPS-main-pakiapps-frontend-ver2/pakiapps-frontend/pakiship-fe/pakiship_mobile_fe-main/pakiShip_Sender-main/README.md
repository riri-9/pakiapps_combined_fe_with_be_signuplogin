# PakiShip Sender Frontend

This folder now contains the sender frontend code copied out from the unified app.

What is here:

- Sender app shell files like `package.json`, `app.config.ts`, `index.ts`, and Expo config files
- Sender auth flow under `src/features/auth`
- Sender home flow under `src/features/sender` and `src/features/customer`
- Sender-only navigation under `src/navigation`
- Required assets and scripts copied from the active app

Important:

- The original intended output is still preserved in `pakiShip_Signup-main`
- This folder is a separate sender-focused workspace for easier access and future cleanup
- The unified app was not removed or replaced
